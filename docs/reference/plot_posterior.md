# Plot fitted posterior summaries

<div class="ref-description section level2">

Plots mean posterior probabilities for each component over the
coefficient grid.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
plot_posterior(object)
```

</div>

</div>

<div class="section level2">

## Arguments

-   object:

    A `vcmoe` object.

</div>

<div class="section level2">

## Value

A `ggplot` object.
